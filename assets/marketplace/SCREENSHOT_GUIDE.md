## üìù Step-by-Step Creation Process

### Step 1: Prepare Your Miro Board

1. Create a new Miro board for screenshots
2. Import a professional floor plan or blueprint
   - Use royalty-free images from:
     - https://unsplash.com (search "blueprint" or "floor plan")
     - https://www.freepik.com
     - Your own architectural drawings
3. Set up realistic measurements:
   - Calibrate properly
   - Create 5-6 measurements
   - Use realistic distances
   - Show different unit types

### Step 2: Take Base Screenshots

1. **Open MeasureMint in Miro**
2. **Adjust your browser window** to show:
   - Clear view of the panel
   - Enough board space
   - Clean, professional layout

3. **Take screenshots** showing:
   - Overview (panel + board with measurements)
   - Calibration modal open
   - Creating a measurement
   - Multiple measurements on a floor plan
   - Unit selector dropdown open

4. **Save as PNG** at high resolution

### Step 3: Design in Figma

1. **Create new Figma file**: 1258 x 706 px
2. **Import screenshot** as background
3. **Add dark overlay** at bottom (20% of height):
   - Rectangle: 1258 x 141 px
   - Position: bottom
   - Color: Black with 70% opacity
   - Or use gradient: Black 100% ‚Üí Transparent 0%

4. **Add text overlay**:
   - Title: 40-50px, Bold, White
   - Subtitle: 24-32px, Regular, White
   - Font: Inter, Poppins, or SF Pro
   - Position: Centered in overlay area
   - Add subtle shadow for readability

5. **Optional enhancements**:
   - Add subtle glow effects
   - Highlight key UI elements
   - Add cursor to show interaction
   - Add arrows or callouts (sparingly)

### Step 4: Export Screenshots

1. **In Figma**:
   - Select frame
   - Click "Export" in right panel
   - Format: PNG
   - Scale: 2x (for retina displays)
   - Export

2. **Naming convention**:
   ```
   screenshot-1-hero.png
   screenshot-2-calibration.png
   screenshot-3-measurement.png
   screenshot-4-use-case.png
   screenshot-5-features.png
   ```

3. **Save to**: `/assets/marketplace/screenshots/`

### Step 5: Optimize File Size

Use online tools if files are too large:
- https://tinypng.com (recommended)
- https://squoosh.app
- https://compressor.io

Target: Under 2MB per screenshot (requirement is 8MB, but smaller is better)
