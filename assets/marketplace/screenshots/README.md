# Marketplace Screenshots

This directory contains the official screenshots for MeasureMint's Miro Marketplace listing.

## Requirements

- **Dimensions:** 1258px × 706px (exactly)
- **Format:** PNG
- **Quantity:** 3-5 screenshots (we use 5)
- **File size:** Under 2MB each
- **Quality:** High resolution, clear text
